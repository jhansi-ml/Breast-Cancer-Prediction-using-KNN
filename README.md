# Breast-Cancer-Prediction-using-KNN
This project uses the K-Nearest Neighbors (KNN) algorithm to classify breast cancer tumors as malignant or benign. 
## 📊 Dataset
- Source: sklearn breast cancer dataset
- Features: 30 numerical features

## ⚙️ Techniques Used
- Data preprocessing
- Feature selection (correlation-based)
- KNN algorithm
- Hyperparameter tuning

## 📈 Results
- Accuracy: ~96%
- Evaluation:
  - Confusion Matrix
  - Precision & Recall

## 📦 Tech Stack
- Python
- NumPy
- Pandas
- Matplotlib
- Scikit-learn
